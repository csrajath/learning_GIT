### TREE
To see the contents or structure of your *.git* file, file that tracks the directories, use the below command.
```git
tree .git 
```
The output of the above command i.e the *.git* file structure would look something like below:
```git
├───hooks
├───info
├───logs
│   └───refs
│       ├───heads
│       └───remotes
│           └───origin
├───objects
│   ├───info
│   └───pack
└───refs
    ├───heads
    ├───remotes
    │   └───origin
    └───tags
```
